# AI Powered Thumbnail Generator

An AI-powered web application that generates professional YouTube thumbnails using AI.

## Features

- JWT Authentication
- AI Thumbnail Generation
- Groq AI Integration
- MongoDB Atlas
- Responsive UI
- Thumbnail Library
- Community Feed

## Tech Stack

### Frontend
- React.js
- Tailwind CSS
- Axios

### Backend
- Node.js
- Express.js
- MongoDB
- JWT
- Bcrypt

### AI
- Groq API
- Pollinations AI

## Installation

### Clone Repository

```bash
git clone https://github.com/Premcoder123/AI-Powered-Thumbnail-Generator.git
```

### Install

```bash
cd client
npm install

cd ../server
npm install
```

### Run

Backend

```bash
npm run dev
```

Frontend

```bash
npm run dev
```

## Environment Variables

Create a `.env` inside the `server` folder.

```
MONGO_URI=
JWT_SECRET=
GROQ_API_KEY=
CLIENT_URL=http://localhost:5173
PORT=5000
```
